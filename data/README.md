# Data

The datasets used in this project are not included in this repository due to file size and licensing considerations.

This workflow is based on a practical exercise involving census data and spatial analysis.

## Data Sources

To reproduce the workflow, users should obtain:

- Census data (e.g. employment and unemployment figures)
- Spatial datasets (e.g. ward boundaries, accessibility data)

These may be obtained from:
- Course practical materials
- Official sources such as NISRA or ONS

## Instructions

Place the input data files in this folder before running the script.

You may need to update file paths in the script depending on your data location.
